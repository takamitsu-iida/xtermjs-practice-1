# ssh xterm.js

[xterm.js](https://github.com/xtermjs/xterm.js/)

## install

install node modules.

```bash
npm install
```

copy config.sample.json to config.json and edit it.

```bash
copy config.sample.json config.json
```

run the server.

```bash
node server.js
```

on your browser, access to localhost:8888
